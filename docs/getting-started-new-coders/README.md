# Getting Started with OneUnlimited | New Coders

Welcome to OneUnlimited! This guide will help you get set up and start using OneUnlimited to build your first project.

## What You'll Need

Before you begin, make sure you have the following:

-   **VS Code:** A free, powerful code editor.
    -   [Download VS Code](https://code.visualstudio.com/)
-   **Development Tools:** Essential software for coding (Homebrew, Node.js, Git, etc.).
    -   Follow our [Installing Essential Development Tools](installing-dev-essentials.md) guide to set these up with OneUnlimited's help (after getting setup here)
    -   OneUnlimited will guide you through installing everything you need
-   **OneUnlimited Projects Folder:** A dedicated folder for all your OneUnlimited projects.
    -   On macOS: Create a folder named "OneUnlimited" in your Documents folder
        -   Path: `/Users/[your-username]/Documents/OneUnlimited`
    -   On Windows: Create a folder named "OneUnlimited" in your Documents folder
        -   Path: `C:\Users\[your-username]\Documents\OneUnlimited`
    -   Inside this OneUnlimited folder, create separate folders for each project
        -   Example: `Documents/OneUnlimited/workout-app` for a workout tracking app
        -   Example: `Documents/OneUnlimited/portfolio-website` for your portfolio
-   **OneUnlimited Extension in VS Code:** The OneUnlimited extension installed in VS Code.

-   Here's a [tutorial](https://www.youtube.com/watch?v=N4td-fKhsOQ) on everything you need to get started.

## Step-by-Step Setup

Follow these steps to get OneUnlimited up and running:

1. **Open VS Code:** Launch the VS Code application. If VS Code shows "Running extensions might...", click "Allow".

2. **Open Your OneUnlimited Folder:** In VS Code, open the OneUnlimited folder you created in Documents.

3. **Navigate to Extensions:** Click on the Extensions icon in the Activity Bar on the side of VS Code.

4. **Search for 'OneUnlimited':** In the Extensions search bar, type "OneUnlimited".

5. **Install the Extension:** Click the "Install" button next to the OneUnlimited extension.

6. **Open OneUnlimited:** Once installed, you can open OneUnlimited in a few ways:
    - Click the OneUnlimited icon in the Activity Bar.
    - Use the command palette (`CMD/CTRL + Shift + P`) and type "OneUnlimited: Open In New Tab" to open OneUnlimited as a tab in your editor. This is recommended for a better view.
    - **Troubleshooting:** If you don't see the OneUnlimited icon, try restarting VS Code.
    - **What You'll See:** You should see the OneUnlimited chat window appear in your VS Code editor.

## Setting up OpenRouter API Key

Now that you have OneUnlimited installed, you'll need to set up your OpenRouter API key to use OneUnlimited's full capabilities.

1.  **Get your OpenRouter API Key:**
    -   [Get your OpenRouter API Key](https://openrouter.ai/)
2.  **Input Your OpenRouter API Key:**
    -   Navigate to the settings button in the OneUnlimited extension.
    -   Input your OpenRouter API key.
    -   Select your preferred API model.
        -   **Recommended Models for Coding:**
            -   `anthropic/claude-3.5-sonnet`: Most used for coding tasks.
            -   `google/gemini-2.0-flash-exp:free`: A free option for coding.
            -   `deepseek/deepseek-chat`: SUPER CHEAP, almost as good as 3.5 sonnet
        -   [OpenRouter Model Rankings](https://openrouter.ai/rankings/programming)

## Your First Interaction with OneUnlimited

Now you're ready to start building with OneUnlimited. Let's create your first project folder and build something! Copy and paste the following prompt into the OneUnlimited chat window:

```
Hey OneUnlimited! Could you help me create a new project folder called "hello-world" in my OneUnlimited directory and make a simple webpage that says "Hello World" in big blue text?
```

**What You'll See:** OneUnlimited will help you create the project folder and set up your first webpage.

## Tips for Working with OneUnlimited

-   **Ask Questions:** If you're unsure about something, don't hesitate to ask OneUnlimited!
-   **Use Screenshots:** OneUnlimited can understand images, so feel free to use screenshots to show him what you're working on.
-   **Copy and Paste Errors:** If you encounter errors, copy and paste the error messages into OneUnlimited's chat. This will help him understand the issue and provide a solution.
-   **Speak Plainly:** OneUnlimited is designed to understand plain, non-technical language. Feel free to describe your ideas in your own words, and OneUnlimited will translate them into code.
